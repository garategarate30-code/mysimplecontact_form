PHP Contact Form CRUD

This is a simple PHP project to Create, Read, Update, and Delete contacts using MySQL.

Requirements:

* XAMPP (Apache + MySQL)
* Web browser

Setup:

1. Copy 'contact-crud' folder to XAMPP's htdocs folder.
2. Start Apache and MySQL in XAMPP.
3. Import 'contact_db.sql' in phpMyAdmin.
4. Open in browser: (http://localhost/contact-crud/index.php)
